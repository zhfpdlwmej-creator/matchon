# matchon 개발 일정표

가정: 풀스택 1~2인, 주 5일. MVP 6주 출시 목표.

## 마일스톤 한눈에
| 단계 | 기간 | 산출물 |
|------|------|--------|
| M0 셋업 | 1주차 | 프로젝트 골격·DB·인증 |
| M1 코어 | 2~3주차 | 팀/일정/참석 |
| M2 보강 | 4주차 | 현황/포지션/댓글/알림 |
| M3 안정화 | 5주차 | 통계/회비/QA |
| M4 출시 | 6주차 | 배포/베타 운영 |

## 주차별 상세 (MVP)

### 1주차 — 환경 & 인증
- [x] Gradle/Spring Boot 3 + JSP + MariaDB + Flyway 셋업
- [x] 엔티티/마이그레이션(V1) 7테이블
- [x] JWT 발급/필터, SecurityConfig
- [x] 카카오 OAuth 로그인 + mock 폴백 + 닉네임 설정

### 2주차 — 팀
- [x] 팀 생성/초대코드/가입, 권한(LEADER/MANAGER/MEMBER)
- [x] 팀원 목록/권한 변경, 내 팀 전환
- [x] JSP: login/welcome/teams/profile, 공통 레이아웃

### 3주차 — 일정 & 참석(핵심)
- [x] 일정 CRUD + 월별 달력 + 권한 가드
- [x] 참석/불참/미정 토글, unique 보장
- [x] 홈: 가장 가까운 경기 + 빠른 참석
- [x] JSP: schedule/list, home

### 4주차 — 현황 & 알림
- [x] 참석 현황 집계(인원·명단·포지션별)
- [x] 일정 댓글
- [x] 알림 추상화(log) + 일정등록 알림 + 리마인드 스케줄러(D-1/3h/30m)
- [x] JSP: schedule/detail
- [ ] 카카오 알림톡 채널/템플릿 신청(승인 리드타임 시작)

### 5주차 — 통계 & 회비 & QA
- [x] QueryDSL 출석률 통계(이번달/3개월/전체)
- [x] 회비 납부 체크/미납 조회
- [x] 인원부족 알림, 관리자 화면
- [ ] 크로스브라우저/모바일 QA, 권한·경계 테스트

### 6주차 — 출시
- [ ] 운영 DB/도메인/HTTPS, 환경변수 분리
- [ ] Dockerfile 배포(PaaS), 모니터링/로그
- [ ] 1~2개 동호회 베타 온보딩, 피드백 수집

## 출시 후 (V1, +4~8주)
- 알림톡 실연동, 출석왕 랭킹, 회비 장부, 카톡 공유 카드, 프로필 보강
