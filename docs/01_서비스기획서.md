# matchon 서비스 기획서

## 1. 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | matchon (매치온) |
| 한 줄 정의 | 카카오톡과 함께 쓰는 축구·풋살 동호회 **출석 관리 보조 서비스** |
| 주 사용자 | 20~100명 규모의 축구/풋살 동호회 |
| 핵심 가치 | 단톡방에서 흩어지는 "참석 인원 집계"를 버튼 한 번으로 끝낸다 |
| 플랫폼 | 모바일 우선 반응형 웹 (설치 불필요, 카카오 로그인) |

### 포지셔닝 — "대체"가 아니라 "보조"
matchon 은 카카오톡 단톡방을 대체하지 않습니다. 소통·잡담·공지는 그대로 단톡방에서 하고,
matchon 은 **"이번 경기 누가 와?"** 라는 단 하나의 반복 작업만 떼어내 자동화합니다.

- 단톡방 ❌ 대체 → 채팅/커뮤니티 기능 없음
- 단톡방 ⭕ 보조 → 초대코드를 단톡방에 공유, 알림도 카카오로 전달

## 2. 문제 정의 (Pain Point)

동호회 총무/운영진의 반복 고통:
1. 단톡방에 "이번 주 일요일 경기 참석 ㅇ/�x 남겨주세요" → 답글이 위로 밀려 집계 불가
2. 누가 참석인지 손으로 세고, 인원 부족하면 다시 독려
3. 회비 납부자/미납자 수기 관리
4. 포지션(GK/DF/MF/FW) 밸런스 안 맞아 당일 우왕좌왕
5. 출석 잘 하는 사람/안 하는 사람 데이터가 없음

## 3. 해결 방안

| 고통 | matchon 해결 |
|------|--------------|
| 참석 집계 | 참석/불참/미정 버튼 → 실시간 자동 집계 |
| 독려 | 일정등록·하루전·3시간전·30분전 자동 카카오 알림 + 인원부족 알림 |
| 회비 | 일정별 납부/미납 체크, 미납자 조회 |
| 포지션 | 참석 현황에서 포지션별 인원 자동 표시 |
| 데이터 | 회원별 이번달/최근3개월/전체 출석률 통계 |

## 4. 사용자 & 권한

| 권한 | 설명 | 주요 권한 |
|------|------|-----------|
| 팀장(LEADER) | 팀 생성자, 최고 관리자 | 일정 CRUD, 권한 변경, 팀 설정, 회비 관리 |
| 운영진(MANAGER) | 팀장이 위임 | 일정 CRUD, 회비 관리, 알림 설정 |
| 일반회원(MEMBER) | 참석 체크 위주 | 참석 체크, 댓글, 내 통계 조회 |

## 5. 핵심 사용자 시나리오

### 5-1. 운영진 — 경기 등록
1. 카카오 로그인 → 팀 선택
2. 일정 탭 → ＋ → 경기명/날짜/시간/장소/참가비/메모 입력 → 저장
3. 저장 즉시 팀원에게 "새 일정 등록" 카카오 알림 발송

### 5-2. 회원 — 참석 체크
1. 단톡방에 공유된 링크 또는 알림 클릭 → 홈
2. 가장 가까운 경기 카드에서 **[참석]** 버튼 탭 → 끝
3. 하루 전/3시간 전/30분 전 리마인드 자동 수신

### 5-3. 총무 — 당일 회비
1. 일정 상세 → 참석자 명단에서 납부 확인 버튼 탭
2. 미납자 한눈에 확인

## 6. 기능 범위 — MVP / V1 / V2

### MVP (4~6주, 출시 목표)
- 카카오 로그인 + 닉네임 설정
- 팀 생성/초대코드 가입/권한
- 경기 일정 CRUD + 월별 달력
- 참석/불참/미정 + 실시간 현황(인원·명단·포지션별)
- 카카오 알림(일정등록/D-1/3h/30m) — *MVP 는 발송 추상화 + 로그, 실제 알림톡은 채널 승인 후*
- 일정 댓글

### V1 (출시 후 1~2개월)
- 출석률 통계(이번달/최근3개월/전체)
- 인원부족 자동 알림
- 회비 관리(참가비 설정/납부 체크/미납자 조회)
- 카카오 알림톡 실연동(비즈채널 + 템플릿 승인)
- 등번호/프로필 보강

### V2 (확장)
- 팀 운영비/회비 장부(월별 정산, 잔액)
- 자동 팀 나누기(포지션·출석률 기반 밸런싱)
- 경기 결과/MOM 기록, 개인 스탯
- 용병(게스트) 초대 링크
- 다중 종목(풋살/축구/족구) 템플릿, 정기 일정 반복 생성
- 푸시(웹푸시/PWA), 캘린더(iCal) 연동

## 7. 비범위 (Out of Scope)
- 실시간 채팅/커뮤니티 (→ 카카오톡 사용)
- 결제(PG) 연동 — 회비는 "체크"만, 실제 송금은 외부
- 경기 매칭/매칭 상대 구하기

## 8. 성공 지표 (KPI)
- 팀당 주간 활성 참석 응답률 ≥ 70%
- 일정당 평균 참석 응답까지 걸리는 시간 ↓
- 운영진의 "집계 수작업 시간" 0 에 수렴
- 4주 리텐션(팀 단위) ≥ 60%

## 9. 추가 제안 (실제 운영자 관점)
1. **정기 일정 자동 생성** — "매주 일요일 8시"를 한 번 등록하면 매주 자동 생성 (V2)
2. **용병 모집 링크** — 인원 부족 시 외부 공유용 1회성 참석 링크 (V2)
3. **노쇼 패널티/벌금 룰** — 불참 변경 마감 시간, 노쇼 카운트 (V2)
4. **출석왕 랭킹** — 월간 출석 랭킹으로 동기부여 (V1~V2)
5. **카카오톡 공유 카드** — 일정/현황을 카톡 공유 메시지로 (V1)
