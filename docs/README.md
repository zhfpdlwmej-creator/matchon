## 핵심 요약
- **컨셉**: 카카오톡을 대체하지 않는 출석 관리 *보조* 서비스
- **스택**: Java 21 · Spring Boot 3.3 · Spring Security + JWT · JPA + QueryDSL · MariaDB · Flyway · JSP
- **MVP 본질**: 카카오 로그인 → 팀/일정 → **참석/불참/미정 원터치 + 실시간 집계 + 카카오 알림**
