# matchon 화면 설계서

모바일 우선(360~430px) 반응형. 하단 고정 5탭 네비게이션. 상단 그린 헤더.

## 0. 화면 목록 & 네비게이션 맵

```
[로그인] ──카카오──▶ (최초) [닉네임 설정(welcome)]
                         │
                         ▼
                    [내 팀(teams)] ──선택/생성/가입──▶ [홈]
                                                       │
   ┌───────────── 하단 탭 ─────────────┐
   홈   일정      팀원      통계     내정보
   │    │         │         │        │
 [홈] [일정목록]  [팀원관리] [통계]  [내정보]
        │          
   [일정 상세/참석관리]      (운영진) [관리자]
```

## 1. 로그인 (`/login`)
- 그린 풀스크린, 중앙 ⚽ 로고 + "matchon"
- 카피: "축구·풋살 동호회 출석 관리 / 카카오톡과 함께 쓰는 출석 보조 서비스"
- **[카카오로 시작하기]** 버튼(노란색) → `/auth/kakao`
- 에러 시 상단 알림 배너

## 2. 닉네임 설정 (`/welcome`) — 최초 1회
- "환영합니다 👋"
- 닉네임 입력(최대 20자, 카카오 닉네임 프리필)
- 포지션 선택칩 GK/DF/MF/FW (선택)
- **[시작하기]** → 홈

## 3. 내 팀 (`/teams`)
- 가입한 팀 카드 리스트(팀명/소개/멤버수/내 권한) → 탭 시 해당 팀 홈
- "새 팀 만들기" 폼(팀명/소개)
- "초대코드로 가입" 폼(6자리 코드)
- 팀 없으면 빈 상태 일러스트

## 4. 홈 (`/team/{teamId}`)  — 탭: 홈
```
┌────────────────────────────┐
│ ⚽ FC 챔피언스 ▾      [팀장] │  헤더
├────────────────────────────┤
│ 가장 가까운 경기            │
│ ┌────────────────────────┐ │
│ │ 6월 20일 (일)          │ │
│ │ FC 챔피언스 vs FC 드림  │ │
│ │ ⏰ 20:00~22:00 · 📍잠실 │ │
│ │ 💰 참가비 5,000원       │ │
│ │  [참석12][불참3][미정5] │ │  실시간 집계
│ │  [ 참석 ][ 불참 ][미정] │ │  내 상태 토글
│ │      자세히 보기 →      │ │
│ └────────────────────────┘ │
│ [📅전체일정]   [👥팀원보기] │
├────────────────────────────┤
│ 🏠   📅   👥   📊   ⚙️       │  하단 탭
└────────────────────────────┘
```
- 예정 경기 없으면 빈 상태(운영진엔 "일정 등록하러 가기")

## 5. 일정 목록 (`/team/{teamId}/schedules`) — 탭: 일정
- 상단 **월별 달력**: ‹ 2026년 6월 › / 일정 있는 날 초록 점 → 탭 시 상세
- 하단 "이번 달 일정" 카드 리스트(날짜/경기명/시간/장소), 지난 경기는 흐리게
- 운영진: 우하단 **＋ FAB** → 등록 모달(경기명/날짜/시작·종료/장소/참가비/메모)
- 모달은 수정/삭제 겸용

## 6. 일정 상세 · 참석 관리 (`/team/{teamId}/schedule/{id}`)
- 경기 정보 카드(날짜/시간/장소/참가비/메모)
- **내 참석 여부**: [참석][불참][미정] 토글(선택 색상 강조)
- **참석 현황**: 참석/불참/미정 박스 + 포지션별 칩(GK n · DF n · MF n · FW n)
- 참석자 명단(포지션 배지 + 닉네임 + 회비 납부/미납 / 운영진은 납부 토글 버튼)
- 불참·미정 명단
- **댓글**: 목록 + 입력창("10분 늦습니다", "주차 가능한가요?")

## 7. 팀원 관리 (`/team/{teamId}/members`) — 탭: 팀원
- 초대코드 카드(큰 코드 + [복사] / 운영진 [재발급]) — "단톡방에 공유하세요"
- 팀원 리스트(포지션 배지 + 닉네임 + 권한)
- 팀장: 권한 드롭다운(팀장/운영진/회원)

## 8. 통계 (`/team/{teamId}/stats`) — 탭: 통계
- 회원별 출석률(전체 참석률 순 정렬)
- 각 행: 순위 · 포지션 · 닉네임 · n/m경기 · 진행바(전체율) · 이번달/최근3개월/전체 %

## 9. 내 정보 (`/profile`) — 탭: 내정보
- 닉네임/포지션 수정
- 내 팀 목록 → 이동, [+ 팀 추가/가입]
- 로그아웃

## 10. 관리자 (`/team/{teamId}/admin`) — 운영진 진입
- 인원부족 알림 기준 인원 설정(0=미사용)
- 알림 발송 이력(타입/메시지/시각/발송여부)
- (비운영진 접근 시 안내 문구)

## 11. 공통 UI 규칙
- 컬러: 메인 그린 `#1a7f37`, 카카오 옐로 `#fee500`, 참석=그린/불참=레드/미정=앰버
- 포지션 배지 색: GK 골드 · DF 블루 · MF 그린 · FW 레드
- 터치 타깃 ≥ 44px, 한 화면 한 액션, 빈 상태/로딩 상태 항상 제공
- 모든 데이터는 AJAX(`/api`)로 비동기 로드 → 새로고침 없이 실시간 반영
